# 総則

## 届出 [a:article1]
[項:base]
従業員は、必要な事項を会社に届け出なければならない。

[項:method]
{{参照:base|相対}}に基づく届出は、書面又は電磁的方法により行う。
- [i:fee] 料金を支払うこと。
- [i:park] 区画を守ること。
- {{ref:park|relative}}を順守すること。

## 適用 [a:article2]
{{参照:article1|相対}}を準用する。
